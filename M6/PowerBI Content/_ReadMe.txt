Power BI Demo Files
*******************

The Report(s) in this folder have a simple Excel data-source 
provided in the Globomantics.xlsx file.

You will need to change the file-path within the pbix data-source 
settings, to match the path on your own machine/environment.

The data-source has been created so that refreshes of the 
data will automatically 'shift' any date-based values.
